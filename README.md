# dewarp_video

Python scripts to dewarp videos (e.g., GoPro)

## Requirements

- Python (>= 2.7 should work fine; not sure about 3.X)
- opencv (http://opencv.org/)
- progressbar (https://code.google.com/p/python-progressbar/)
